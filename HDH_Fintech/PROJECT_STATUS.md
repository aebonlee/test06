## 🎓 빠른 시작 (Quick Start)

### Claude Code로 작업 시작하기

1. **현재 상황 파악**
   ```
   이 파일 (PROJECT_STATUS.md)을 먼저 읽어 현재 진행 상황 확인
   ```

2. **디렉토리 구조 확인**
   ```
   0_프로젝트_기획/ → 기획 문서
   1_프로토타입_제작/ → 개발 코드
   .claude/ → Claude Code 설정
   Web_ClaudeCode_Bridge/ → Inbox/Outbox
   ```

3. **작업 시작**
   ```
   현재는 백엔드 개발 단계
   1_프로토타입_제작/ 폴더에서 작업 시작
   ```
