## 🎯 서비스 소개 본문

### 오프닝 (선언형 + 비전 제시형)

**HDH Fintech 웹사이트는 HDH 핀테크 솔루션 운영을 체계적으로 지원하는 회원 전용 웹사이트입니다.**

10년간 검증된 프롭펌 헷지 트레이딩 전략으로 안정적인 수익 창출을, AI 자동화 시스템으로 효율적인 운영을, 체계적인 가이드로 누구나 따라할 수 있는 실전 트레이딩을 제공합니다.

---

### 💡 HDH 솔루션이란? (핵심만 요약)

HDH는 프롭펌 챌린지와 실거래 계좌를 활용한 **양방향 헷지 전략**으로 10년간 안정적인 수익을 검증해온 트레이딩 솔루션입니다.

**핵심 전략:**
- **1차 챌린지 단계:** 프롭펌 챌린지 계좌 + 보험금 계좌 동시 운영으로 합격 시 펀딩 확보, 탈락 시에도 수익 발생
- **2차 펀딩계좌 트레이딩 단계:** 펀딩 계좌 2개 확보 후 양방향 헷지로 80% 정산 수령

자세한 솔루션 원리와 수익 구조는 웹사이트 내 **"솔루션 이해하기"** 메뉴에서 확인하실 수 있습니다.

**"처음이어도 괜찮습니다. 체계적인 교육으로 전문가처럼."**

HDH Fintech 웹사이트는 초보자도 쉽게 따라할 수 있도록 단계별 학습 콘텐츠를 제공합니다.

**제공되는 학습 자료:**
- **솔루션 이해하기:** HDH의 핵심 원리, 10년 검증 실적, 헷지 전략, 수익 구조
- **아카데미 교육 (Day 1-3):**
  - Day 1: 솔루션 이론 (프롭펌 챌린지, 금선물 CFD, 헷지 전략)
  - Day 2: 플랫폼 사용법 (MT5 설정, 자동화 시스템)
  - Day 3: 실전 시뮬레이션 (모의거래, 최종 테스트)
- **매뉴얼:** 거래 가이드, 계좌 설정 방법, 비상 대응

### 2️⃣ 준비 단계: 체계적인 온보딩 프로세스

**"5단계 준비 프로세스로 실전 투입 준비 완료."**

HDH 운영을 시작하기 전, 반드시 거쳐야 할 5단계 준비 과정을 안내합니다.

**준비 5단계:**
1. 솔루션 이해하기
2. 아카데미 교육 수강하기
3. 모의거래 실행하기
4. 추천인 코드 받기
5. 매뉴얼 숙지하기

각 단계마다 상세한 체크리스트와 가이드가 제공되어 누락 없이 완벽하게 준비할 수 있습니다.

### 3️⃣ 실행 단계: 일일 운영 프로세스 가이드

**"매일 무엇을 해야 하는지 명확하게. 체크리스트로 실수 방지."**

HDH 운영은 복잡하지 않습니다. 웹사이트에서 제공하는 일일 체크리스트를 따라 진행하면 됩니다.

**1차: 챌린지 단계 프로세스 (5단계)**
1. 거래 시간 확인 + 알람 예약
2. 시스템 확인 (MT5, 챌린지/보험금 계좌)
3. USDT 매입 및 송금
4. 챌린지 + 보험금 계좌 동시 거래 실행
5. 일일 결과 확인 및 기록

**2차: 펀딩계좌 트레이딩 단계 프로세스 (5단계)**
- 펀딩 계좌 2개 확보 시 실행
- 양방향 헷지 후 80% 정산 수령 과정 안내

각 단계마다 **완료/미완료** 상태를 실시간으로 확인하며, 링크를 통해 관련 기능에 바로 접근할 수 있습니다.

### 4️⃣ 관리 단계: 나만의 운영 환경 구축

**"설정부터 실적 추적까지, 모든 것을 한 곳에서."**

HDH Fintech 웹사이트는 개인화된 운영 관리 기능을 제공합니다.

**My 설정 (7가지 관리 기능):**
- 거래 시작일 설정
- 누적손익 목표 설정
- **계좌 관리:** 계좌 현황, 새 계좌 추가, 연결된 계좌 관리, 연동 로그
- **API 연결 설정:** API 키 관리, 연결 테스트, 권한 확인, 사용량 통계
- **알림 설정:** 포지션 청산, 계좌 연동, 동기화 오류, API 오류 알림
- **데이터 관리:** 거래 내역 다운로드, 데이터 삭제, 동기화 설정
- **프로필 관리:** 회원 정보 수정, 비밀번호 변경, 회원 탈퇴

**My 누적손익:**
- 월/연 누적 손익 실시간 추적
- 목표 달성률 확인
- 레벨 및 랭킹 시스템

**My 손익 발생일지:**
- 일자별 거래 기록 및 수익/손실 추적
- 패턴 분석 및 개선점 도출

### 5️⃣ 자동화 단계: AI 시스템으로 실행 자동화

**"거래 실행을 AI가 대신. 정확도는 높이고, 시간은 절약."**

HDH 운영의 핵심인 **거래 실행 프로세스를 자동화**합니다. 수동 조작의 오차와 실수를 제거하고, 50ms 이하의 초정밀 동기화로 완벽한 양방향 헷지를 구현합니다.

**AI 자동화 시스템 v1.1 주요 기능:**

**1차 챌린지 단계 자동화:**
- **원클릭 동시 진입:** 챌린지 계좌 + 보험금 계좌를 1회 클릭으로 동시 체결 (50ms 이하 오차)
- **음성 명령 지원:** "매수 0.1랏" 등 음성으로 즉시 매매
- **조건 도달 시 자동 청산:** TP/SL 도달 시 자동 청산

**2차 펀딩 트레이딩 단계 자동화:**
- **SL 자동 감지:** 펀딩 계좌 A 또는 B의 손절 발생 실시간 모니터링
- **반대 계좌 즉시 청산:** 한쪽 손절 감지 시 반대 계좌 자동 시장가 청산
- **초정밀 타이밍:** 20-80ms 이내 오차로 양방향 헷지 수익 확보
- **음성 명령 지원:** 수동 개입 시에도 음성으로 즉시 실행

**제공 자료:**
- 프로그램 다운로드
- 설치 및 MT5 연동 가이드
- 사용법 동영상

**레벨 시스템:**
