## 1. 🔔 알림 설정

### 왜 필요한가?

**문제점**:
- 중요한 거래 결과를 놓칠 수 있음
- 목표 달성 시점을 모름
- 계좌 연동 오류를 늦게 발견
- 손절이 발생해도 모를 수 있음

**해결책**:
- 실시간 알림으로 즉시 확인
- 사용자가 원하는 알림만 선택
- 알림 방식 선택 (웹/이메일/SMS)

---

### 1.1 알림 종류 (총 11가지)

#### 그룹 1: 거래 알림 (3개)

##### 1) 포지션 청산 알림

**언제 발생**: 매매 포지션이 청산되었을 때

**알림 내용**:
```
💰 포지션 청산
챌린지 계좌에서 포지션이 청산되었습니다.

손익: +₩1,950,000
심볼: XAUUSD (금 선물)
진입가: 2650
청산가: 2671.5
LOT: 6.8

[상세 보기]
```

**중요도**: ⭐⭐⭐⭐⭐ (필수)

**기본 설정**: ON (체크됨)

---

##### 2) 손익 목표 달성 알림

**언제 발생**:
- 일일 목표 달성 시
- 월간 목표 달성 시
- 연간 목표 달성 시

**알림 내용**:
```
🎉 목표 달성!
월 누적손익 목표를 100% 달성했습니다!

현재 손익: ₩5,320,000
목표 금액: ₩5,320,000

[대시보드 보기]

**중요도**: ⭐⭐⭐⭐

**기본 설정**: ON (체크됨)

##### 3) 손절 발생 알림

**언제 발생**: 손절(Stop Loss)이 발생했을 때

**알림 내용**:

##### 5) 데이터 동기화 오류 알림

**언제 발생**:
- 계좌 데이터 가져오기 실패
- 5회 이상 연속 동기화 실패

**알림 내용**:
```
⚠️ 동기화 오류
챌린지 계좌 데이터 동기화에 실패했습니다.

[고객센터]

**기본 설정**: ON (체크됨)

##### 6) 잔고 변동 알림 (선택 사항)

**언제 발생**: 계좌 잔고가 일정 금액 이상 변동했을 때

**설정 옵션**:
- 변동 금액: ₩100,000 이상 / ₩500,000 이상 / ₩1,000,000 이상

**알림 내용**:
```
💵 잔고 변동
챌린지 계좌의 잔고가 변동했습니다.

이전 잔고: $3,200
현재 잔고: $3,580
변동액: +$380 (₩503,000)

[계좌 현황 보기]
```

**중요도**: ⭐⭐ (선택)

**기본 설정**: OFF (체크 안 됨)

**이유**: 너무 자주 발생할 수 있어 기본은 OFF

### 1.2 알림 방식 (3가지)

#### 1) 웹 푸시 알림

**특징**:
- 브라우저 알림
- 실시간 전송 (즉시)
- 사이트를 닫아도 수신 가능

**설정 방법**:
```
1. "알림 허용" 버튼 클릭
2. 브라우저에서 "허용" 클릭
3. 완료
```

**알림 예시**:
```
┌─────────────────────────────┐
│ HDH Fintech                  │
├─────────────────────────────┤
│ 💰 포지션 청산               │
│ 챌린지 계좌 +₩1,950,000     │
│                              │
│ 1분 전                       │
└─────────────────────────────┘
```

**장점**:
- 즉시 확인 가능
- 무료
- 클릭 시 해당 페이지로 이동

**기본 설정**: ON (체크됨)

---

#### 2) 이메일 알림

**특징**:
- 등록된 이메일로 전송
- 상세 내용 포함
- 기록 보관 가능

**발송 시점**:
- 즉시 발송 (중요 알림)
- 1시간마다 요약 발송 (일반 알림)

**이메일 예시**:
```
제목: [HDH Fintech] 포지션 청산 알림

안녕하세요, 홍길동님!

챌린지 계좌에서 포지션이 청산되었습니다.

━━━━━━━━━━━━━━━━━━━━━━━━
거래 정보
━━━━━━━━━━━━━━━━━━━━━━━━
심볼: XAUUSD (금 선물)
진입가: 2650
청산가: 2671.5
LOT: 6.8
손익: +₩1,950,000 ✅
━━━━━━━━━━━━━━━━━━━━━━━━

[거래 내역 확인하기]

이 메일은 알림 설정에 따라 자동 발송되었습니다.
알림 설정 변경: 설정 > 알림 설정
```

**장점**:
- 상세한 정보 확인
- 나중에 검색 가능
- 모바일에서도 확인 가능

**기본 설정**: ON (체크됨)

---

#### 3) SMS 알림 (유료)

**특징**:
- 휴대폰 문자 전송
- 100% 도달률
- 건당 20원

**발송 조건**:
- 중요 알림만 발송 (포지션 청산, 손절, 계좌 연동 해제)
- 하루 최대 10건

**SMS 예시**:
```
[HDH Fintech]
포지션 청산
챌린지 계좌 +₩1,950,000
상세: https://hdh.fi/trade/12345
```

**요금제**:
- 월 1,000원 (월 50건)
- 월 2,000원 (월 100건)
- 월 5,000원 (무제한)

**기본 설정**: OFF (체크 안 됨)

**이유**: 유료 서비스이므로 사용자가 직접 선택

**1시간 단위 요약 발송** (이메일):

## 2. 📊 데이터 관리

### 왜 필요한가?

**문제점**:
- 거래 내역을 외부 분석 도구에서 사용하고 싶음
- 세무 신고용 손익 증빙 필요
- 오래된 데이터로 용량 부족
- 계좌 연동 오류 원인 파악 어려움

**해결책**:
- 데이터 다운로드 (CSV, PDF)
- 오래된 데이터 삭제
- 동기화 설정 관리

---

### 2.1 데이터 다운로드 (3가지)

#### 1) 거래 내역 다운로드 (CSV)
