## 2. 유저 타입별 플로우

### 2.1. 비회원 (방문자)

**목표:** 서비스 이해 및 신뢰 구축

```
1. 랜딩 페이지 진입
   ↓
2. 서비스 소개 확인
   - 10년 검증된 솔루션
   - 월 500만원 안정적 수익
   - 하루 1시간 투입
   - 저위험 자동화 시스템
   ↓
3. 핵심 솔루션 구조 이해
   - 프롭펌 챌린지 시스템
   - 금선물 CFD 거래
   - 헷지 전략
   ↓
4. 추천인 시스템 이해
   - 추천인 코드 필요성
   - 기존 회원의 1:1 가이드
   ↓
5. CTA (Call To Action)
   - [추천인 코드가 있으신가요?] → 회원가입
   - [추천인 코드가 없으신가요?] → 문의하기
```

---

### 2.2. 열람 회원 (무료)

**목표:** 정보 확인 및 활동 회원 전환 유도

```
1. 회원가입 (추천인 코드 입력)
   ↓
2. 이메일/SMS 인증
   ↓
3. 열람 회원 대시보드 접근
   - 솔루션 상세 설명 (전체 공개)
   - 10년 실적 데이터 (일부 공개)
   - 교육 콘텐츠 목차 (미리보기)
   - 커뮤니티 (읽기 전용)
   ↓
4. 활동 회원 전환 안내
   - 필요 조건 확인
   - 투자 여력: ₩5,000만원
   - 적합성 검사 필요
   - 3일 교육 이수 필요
   ↓
5. 결정
   - [활동 회원 신청하기] → 적합성 검사
   - [더 알아보기] → 계속 열람
```

---

### 2.3. 활동 회원 (실제 운영)

**목표:** 안전한 온보딩 → 성공적 운영

#### Step 1: 활동 회원 신청

```
1. 활동 회원 신청서 제출
   - 기본 정보
   - 투자 여력 확인서
   - 금융 투자 경험 (선택)
   ↓
2. 투자 여력 확인
   - 최소 ₩5,000만원 보유 확인
   - 은행 잔고 증명서 또는 자산 증빙
   ↓
3. 적합성 검사
   - 투자 성향 평가
   - 리스크 수용 능력
   - 시간 투입 가능 여부 (하루 1시간)
   ↓
4. 심사 결과
   - [승인] → 교육 단계
   - [보류] → 추가 자료 요청
   - [거절] → 열람 회원 유지
```

#### Step 2: 교육 이수

```
1. 3일 교육 프로그램 시작

   Day 1: 솔루션 이론
   - 프롭펌 챌린지 시스템 이해
   - 금선물 CFD 거래 기초
   - 헷지 전략 원리
   - 리스크 관리 방법

   Day 2: 플랫폼 사용법
   - MT5 플랫폼 설치 및 설정
   - 자동화 시스템 설정
   - 원클릭 거래 실습
   - 모니터링 대시보드 사용법

   Day 3: 실전 시뮬레이션
   - 데모 계좌 거래 실습
   - 하루 3회 거래 타이밍
   - 손절/익절 자동화 확인
   - 비상 상황 대응 매뉴얼
   ↓
2. 교육 이수 테스트
   - 이론 평가 (70점 이상)
   - 실습 평가 (시뮬레이션 성공)
   ↓
3. 수료증 발급
   ↓
4. 활동 회원 정식 등록
```

#### Step 3: 실제 운영 시작

```
1. 초기 설정
   - 프롭펌 챌린지 계좌 개설
   - 보증금 계좌 개설
   - 초기 자금 입금 (₩5,000만원)
   - 자동화 시스템 연동
   ↓
2. 1주차: 모니터링 집중 기간
   - 멘토(추천인)의 1:1 가이드
   - 하루 3회 거래 (오전/오후/저녁)
   - 실시간 수익 확인
   - 질문 및 피드백
   ↓
3. 2주차 이후: 자립 운영
   - 독립적 거래 실행
   - 주간 수익 리포트 확인
   - 커뮤니티 참여 (경험 공유)
   ↓
4. 월간 정산
   - 수익 정산 및 출금
   - 월 평균 ₩500만원 내외
```

## 7. 유저 여정 요약

| 단계 | 예상 소요 시간 | 주요 활동 |
