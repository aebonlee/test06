1.3 WTG 홈페이지 사용법

WTG 플랫폼의 주요 기능 사용법을 안내합니다.

WTG 대시보드 이해하기

대시보드에서 계좌 잔고와 거래 내역을 확인할 수 있습니다.
챌린지 진행 상황과 목표 달성률을 모니터링합니다.
일일/총 손실 한도 현황을 실시간으로 확인합니다.

지갑 주소 입력 방법

설정 메뉴에서 지갑 주소 등록 페이지로 이동합니다.
바이낸스에서 복사한 지갑 주소를 붙여넣습니다.
주소 확인 후 저장 버튼을 클릭합니다.

바이낸스 지갑 주소 등록

바이낸스에 로그인하여 지갑 메뉴로 이동합니다.
USDT 입금 주소를 확인합니다.
해당 주소를 WTG에 등록합니다.

Network 주소 설정

Address = 계좌번호 (지갑 주소)
Network = 은행명 (전송 네트워크, 예: TRC-20)
반드시 동일한 네트워크를 선택해야 합니다.

⚠️ 주의사항

지갑 주소와 네트워크가 일치하지 않으면 자산을 영구적으로 잃을 수 있습니다. 반드시 확인 후 진행하세요.
