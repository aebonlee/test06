# User Flow #4: 문의 프로세스 (Inquiry Flow)

> **최종 수정:** 2025-12-09
> **버전:** 1.0

---

## 1. 플로우 개요

**목적:** HDH 솔루션 운영자가 궁금한 점을 문의하고 지원을 받는다.

**시작:** Dashboard에서 문의 메뉴 접근
**종료:** 문의 완료 또는 답변 확인
**예상 소요 시간:** 2-5분 (문의 작성 시)

---

## 2. 정상 플로우 (Normal Flow)

### **[문의 경로 1] Sunny에게 묻기 (AI 챗봇)**

#### Step 1-1: Sunny 챗봇 접근
- **위치:** Dashboard 우측 사이드바 하단
- **표시:** "☀️ Sunny에게 묻기" 위젯

#### Step 1-2: 질문 입력
- **액션:** 텍스트 입력창에 질문 입력
- **플레이스홀더:** "사용법 문의, 문제 해결, 기능 제안, HDH 전략 검토, 트레이딩 멘토링까지! AI 활용 능력 고수 + 공인회계사 + 엑셀러레이터인 Sunny에게 자유롭게 문의하세요!"

**예시 질문:**
- "API 연결이 안 돼요"
- "손익 저널 작성 방법 알려주세요"
- "추천인 코드는 어떻게 받나요?"

#### Step 1-3: 질문 제출
- **액션:** "문의하기" 또는 "Sunny에게 묻기" 버튼 클릭
- **처리:** AI가 질문 분석 후 답변 제공

#### Step 1-4: AI 응답 확인
**응답 예시:**
```
API 연결 문제를 겪고 계시군요. 다음 단계를 확인해주세요:

1. My Settings에서 API 키 확인
   - 올바른 API 키가 입력되어 있나요?
   - [My Settings 바로가기]

2. 계좌 상태 확인
   - 계좌가 활성화되어 있나요?
   - 계좌에 충분한 잔액이 있나요?

3. 네트워크 연결 확인
   - 인터넷 연결이 정상인가요?

📖 상세 가이드: [API 연결 매뉴얼 보기]

도움이 더 필요하시면 고객센터로 문의해주세요.
[고객센터 문의하기]
```

#### Step 1-5: 추가 질문 또는 종료
- **옵션 1:** 추가 질문 입력
- **옵션 2:** 답변 확인 후 종료

---

### **[문의 경로 2] 고객센터 문의 (Customer Center)**

#### Step 2-1: 고객센터 페이지 진입
- **액션:** 하단 푸터 "고객센터" 링크 클릭
- **페이지:** `/contact.html` 또는 고객센터 페이지

#### Step 2-2: 문의 유형 선택
**표시 내용:**
```
📞 고객센터

무엇을 도와드릴까요?

[일반 문의]
- 서비스 이용 방법
- 계정 관련 문의

[기술 지원]
- API 연결 문제
- 프로그램 오류
- 설정 문제

[교육 문의]
- 아카데미 교육 관련
- 추천인 코드 문의

[긴급 문의]
- 즉시 처리가 필요한 문제
```

#### Step 2-3: 문의 작성 폼 진입
- **액션:** 문의 유형 선택
- **표시:** 문의 작성 폼

**입력 항목:**
- 이름 (자동 입력)
- 이메일 (자동 입력)
- 전화번호 (자동 입력)
- 문의 유형 (선택됨)
- 제목 (필수)
- 문의 내용 (필수)
- 첨부 파일 (선택) - 스크린샷, 로그 파일 등

#### Step 2-4: 문의 제출
- **액션:** "문의하기" 버튼 클릭
- **처리:** 담당자에게 문의 내용 전달

**제출 완료 메시지:**
```
✅ 문의가 접수되었습니다!

문의 번호: #2025120901

📌 처리 안내:
- 일반 문의: 1-2일 이내 답변
- 기술 지원: 당일 답변
- 긴급 문의: 2-4시간 이내 답변

답변은 이메일로 발송되며, 고객센터에서도 확인 가능합니다.

[내 문의 내역 보기] [확인]
```

#### Step 2-5: 문의 내역 확인
- **위치:** 고객센터 페이지 내 "내 문의 내역" 탭
- **표시 내용:**
  - 문의 번호
  - 제목
  - 상태 (접수/처리중/완료)
  - 등록일
  - 답변일

**문의 상세 보기:**
- 문의 항목 클릭
- 원본 문의 내용 확인
- 답변 내용 확인 (답변 완료 시)

---

### **[문의 경로 3] 공지사항 확인**

#### Step 3-1: 공지사항 확인
- **위치:** Dashboard 좌측 사이드바 상단 "공지사항" 위젯
- **표시 내용:**
  - 최근 공지사항 3개
  - 공지 날짜 및 내용

#### Step 3-2: 공지사항 상세 보기
- **액션:** 공지사항 항목 클릭
- **표시:** 공지사항 전체 내용 (모달 또는 별도 페이지)

**공지 유형:**
- 📢 중요 공지
- 🔔 시스템 업데이트
- ⚠️ 긴급 공지

## 3. 예외 플로우 (Exception Flows)

### 3-1. 문의 내용 미입력
- **조건:** 필수 항목을 채우지 않고 제출
- **시스템 응답:**
```
❌ 필수 항목을 모두 입력해주세요.

- 제목 (필수)
- 문의 내용 (필수)
```

### 3-2. 첨부 파일 용량 초과
- **조건:** 10MB 이상 파일 첨부 시도
- **시스템 응답:**
```
❌ 첨부 파일이 너무 큽니다.

- 최대 용량: 10MB
- 현재 파일: 15MB

파일을 압축하거나 분할하여 첨부해주세요.
```

### 3-3. AI 챗봇 응답 불가
- **조건:** AI가 질문을 이해하지 못하는 경우
- **시스템 응답:**
```
😅 죄송합니다. 질문을 정확히 이해하지 못했어요.

다음 방법을 시도해보세요:
- 질문을 더 구체적으로 작성해주세요
- 고객센터에 직접 문의하기

[고객센터 문의]
```

## 5. 성공 기준

### 5-1. 기능적 성공
- ✅ 사용자가 AI 챗봇(Sunny)을 통해 즉시 답변을 받을 수 있다
- ✅ 사용자가 고객센터에 문의를 제출할 수 있다
- ✅ 문의 상태를 추적할 수 있다
- ✅ 공지사항을 확인할 수 있다

### 5-2. UX 성공
- ✅ 문의 경로가 명확하고 접근이 쉽다
- ✅ AI 챗봇이 빠르고 정확하게 답변한다
- ✅ 고객센터 문의 작성이 간단하다
- ✅ 답변 확인이 쉽다

### 5-3. 비즈니스 성공
- ✅ 자주 묻는 질문은 AI로 즉시 해결 (고객센터 부담 감소)
- ✅ 복잡한 문의는 고객센터에서 처리
- ✅ 사용자 만족도 향상
