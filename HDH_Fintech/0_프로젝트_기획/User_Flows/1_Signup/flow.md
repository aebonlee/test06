# User Flow #1: 회원가입 (Signup Flow)

> **최종 수정:** 2025-12-09
> **버전:** 1.0

---

## 1. 플로우 개요

**목적:** 신규 사용자가 HDH Fintech에 회원가입하여 HDH 솔루션 운영 준비를 시작한다.

**시작:** 랜딩 페이지에서 "회원가입" 버튼 클릭
**종료:** 계정 생성 완료 + Dashboard 진입
**예상 소요 시간:** 3분

---

## 2. 정상 플로우 (Normal Flow)

### Step 1: 회원가입 페이지 진입
- **액션:** 사용자가 "회원가입" 버튼 클릭
- **페이지:** `/signup`
- **표시 내용:**
  - 페이지 제목: "HDH Fintech 회원가입"
  - 회원가입 폼
  - HDH 솔루션 운영자를 위한 지원 플랫폼 안내

### Step 2: 회원 정보 입력
**필수 입력 항목:**

| 필드 | 타입 | 제약 조건 | 검증 |
|------|------|----------|------|
| 이메일 | email | 필수, 고유값 | 이메일 형식 검증 |
| 비밀번호 | password | 필수, 8자 이상 | 영문+숫자+특수문자 |
| 비밀번호 확인 | password | 필수 | 비밀번호와 일치 |
| 닉네임 | text | 필수, 2-20자 | 중복 검증 |
| 이름 | text | 필수 | 한글/영문 |
| 전화번호 | tel | 필수 | 휴대폰 번호 형식 |

### Step 3: 약관 동의
- **필수 동의:**
  - ✅ 서비스 이용약관 (필수)
  - ✅ 개인정보 처리방침 (필수)
- **선택 동의:**
  - ☑️ 마케팅 정보 수신 (선택)

### Step 4: "회원가입" 버튼 클릭
- **액션:** 서버로 회원가입 요청 전송
- **처리:** 입력 정보 검증 및 계정 생성

### Step 5: 회원가입 완료 + Dashboard 진입
- **페이지:** Dashboard 진입 (`/dashboard`)
- **환영 메시지:**
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎉 HDH Fintech에 오신 것을 환영합니다!

HDH 솔루션 운영을 위한 모든 지원이 준비되어 있습니다.

📌 다음 단계:
1. 솔루션 이해하기
2. 아카데미 교육 수강하기
3. 추천인 코드 신청하기
4. 매뉴얼 숙지하기

[시작하기]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Step 6: Dashboard 확인
- **페이지:** `/dashboard`
- **표시 내용:**
  - 좌측 사이드바: HDH 운영 준비 프로세스 (4단계)
  - 중앙: 주요 콘텐츠 영역
  - 우측 사이드바: 공지사항, 관리 메뉴

## 3. 예외 플로우 (Exception Flows)

### 3-1. 이메일 중복
- **조건:** 이미 존재하는 이메일로 가입 시도
- **시스템 응답:**
```
❌ 이미 가입된 이메일입니다.
[로그인하기] [다른 이메일 사용]
```

### 3-2. 닉네임 중복
- **조건:** 이미 존재하는 닉네임 입력
- **시스템 응답:**
```
❌ 이미 사용 중인 닉네임입니다.
다른 닉네임을 입력해주세요.
```

### 3-3. 비밀번호 불일치
- **조건:** 비밀번호와 비밀번호 확인이 다름
- **시스템 응답:**
```
❌ 비밀번호가 일치하지 않습니다.
```

### 3-4. 비밀번호 강도 부족
- **조건:** 8자 미만 또는 영문+숫자+특수문자 미충족
- **시스템 응답:**
```
❌ 비밀번호가 너무 약합니다.
최소 8자 이상, 영문+숫자+특수문자 포함 필요
```

### 3-5. 필수 약관 미동의
- **조건:** 필수 약관 체크 안 함
- **시스템 응답:**
```
❌ 필수 약관에 동의해주세요.
```

### 3-6. 전화번호 형식 오류
- **조건:** 잘못된 전화번호 형식 입력
- **시스템 응답:**
```
❌ 올바른 전화번호 형식이 아닙니다.
예: 010-1234-5678
```

## 4. 플로우 다이어그램

```
[랜딩 페이지]
    ↓
[회원가입 버튼]
    ↓
[회원가입 폼]
- 이메일 (필수)
- 비밀번호 (필수)
- 비밀번호 확인 (필수)
- 닉네임 (필수)
- 이름 (필수)
- 전화번호 (필수)
    ↓
[약관 동의]
- 서비스 이용약관 (필수)
- 개인정보 처리방침 (필수)
- 마케팅 수신 (선택)
    ↓
[회원가입 버튼]
    ↓
[서버 검증]
    ├─ ❌ 이메일 중복 → 오류
    ├─ ❌ 닉네임 중복 → 오류
    ├─ ❌ 비밀번호 불일치 → 오류
    ├─ ❌ 전화번호 오류 → 오류
    └─ ✅ 검증 통과
         ↓
    [계정 생성]
         ↓
    [환영 메시지]
    - 다음 단계 안내
         ↓
    [Dashboard 진입]
    - 좌측: 준비 프로세스 (4단계)
    - 중앙: 메인 콘텐츠
    - 우측: 공지/관리
```

## 5. 성공 기준

### 5-1. 기능적 성공
- ✅ 사용자가 회원가입 폼을 작성할 수 있다
- ✅ 모든 입력 검증이 작동한다
- ✅ 계정이 생성되고 DB에 저장된다
- ✅ Dashboard에 진입한다
- ✅ 준비 프로세스 메뉴가 표시된다

### 5-2. UX 성공
- ✅ HDH 솔루션 운영자를 위한 플랫폼임이 명확히 안내된다
- ✅ 오류 메시지가 명확하다
- ✅ 다음 단계(준비 프로세스)가 명확히 안내된다

### 5-3. 비즈니스 성공
- ✅ HDH 솔루션 운영 희망자가 쉽게 가입할 수 있다
- ✅ 회원가입 완료 후 다음 단계가 명확하다
