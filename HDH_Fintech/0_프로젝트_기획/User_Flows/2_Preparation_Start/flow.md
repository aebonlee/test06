# User Flow #2: HDH 운영 준비 및 시작 (Preparation & Operation Start Flow)

> **최종 수정:** 2025-12-09
> **버전:** 1.0

---

## 1. 플로우 개요

**목적:** 회원이 HDH 솔루션 운영을 위한 준비 과정을 완료하고, 실제 운영을 시작한다.

**시작:** Dashboard에서 준비 프로세스 시작
**종료:** HDH 솔루션 운영 시작 (모든 준비 완료)
**예상 소요 시간:** 교육 이수 시간에 따라 가변 (수일~수주)

---

## 2. 정상 플로우 (Normal Flow)

### Step 1: Dashboard에서 준비 프로세스 확인
- **페이지:** `/dashboard`
- **표시 내용:** 좌측 사이드바에 준비 프로세스 4단계
  1. HDH 솔루션 이해하기
  2. 아카데미 교육 수강하기
  3. 추천인 코드 신청하기
  4. 매뉴얼 숙지하기

---

### **[단계 1] HDH 솔루션 이해하기**

#### Step 2-1: 솔루션 소개 페이지 진입
- **액션:** "HDH 솔루션 이해하기" 메뉴 클릭
- **페이지:** `/solution-intro`
- **표시 내용:**
  - HDH 솔루션이란? (10년 검증된 프롭펌 헤지 트레이딩 전략)
  - 핵심 특징
  - 수익 구조
  - 운영 방식

#### Step 2-2: 콘텐츠 학습
- **액션:** 사용자가 솔루션 소개 콘텐츠를 읽고 이해
- **결과:** HDH 솔루션에 대한 기본 이해 완료

---

### **[단계 2] 아카데미 교육 수강하기**

#### Step 3-1: 아카데미 소개 확인
- **액션:** "아카데미 교육 수강하기" 메뉴 클릭
- **페이지:** `/academy-intro`
- **표시 내용:**
  - 교육 과정 소개
  - 교육 커리큘럼
  - 교육 기간
  - 수강 신청 버튼

#### Step 3-2: 수강신청 페이지 진입
- **액션:** "수강신청하기" 버튼 클릭
- **페이지:** `/academy-apply`

#### Step 3-3: 수강신청 정보 입력
**필수 입력 항목:**
- 이름 (자동 입력됨)
- 이메일 (자동 입력됨)
- 전화번호 (자동 입력됨)
- 수강 동기
- 현재 트레이딩 경험
- 기대하는 점

#### Step 3-4: 신청 제출
- **액션:** "신청하기" 버튼 클릭
- **처리:** 담당자에게 신청 내용 전달

#### Step 3-5: 신청 완료 확인
- **표시 메시지:**
```
✅ 수강신청이 완료되었습니다!
담당자가 확인 후 1-2일 이내에 연락 드리겠습니다.

📌 안내:
- 교육 일정은 개별 안내 예정
- 교육 완료 후 추천인 코드 신청 가능
```

#### Step 3-6: 교육 수강
- **프로세스:** 담당자 연락 → 교육 일정 조율 → 교육 진행
- **교육 방식:** 온라인/오프라인 (별도 협의)
- **결과:** 교육 이수 완료

---

### **[단계 3] 추천인 코드 신청하기**

#### Step 4-1: 추천인 코드 신청 페이지 진입
- **액션:** "추천인 코드 신청하기" 메뉴 클릭
- **페이지:** `/referral-apply`
- **조건:** 아카데미 교육 이수 필요

#### Step 4-2: 신청 정보 확인
**자동 입력 항목:**
- 이름
- 이메일
- 전화번호
- 교육 이수 여부 (시스템 확인)

**입력 항목:**
- 신청 사유
- HDH 솔루션을 어떻게 알게 되었는지

#### Step 4-3: 신청 제출
- **액션:** "신청하기" 버튼 클릭
- **처리:** 담당자가 교육 이수 여부 확인 후 승인

#### Step 4-4: 승인 완료
- **표시 메시지:**
```
✅ 추천인 코드가 발급되었습니다!

🎯 나의 추천인 코드: HDH2025ABC123

📌 추천인 코드 용도:
- 회원 유입 경로 추적용
- 향후 통계 분석 자료로 활용

💡 참고:
- 추천인 코드는 순수 추적 목적이며, 별도 리워드는 없습니다.
```

**결과:** 추천인 코드 발급 완료 (회원 유입 경로 추적용)

---

### **[단계 4] 매뉴얼 숙지하기**

#### Step 5-1: 매뉴얼 페이지 진입
- **액션:** "매뉴얼 숙지하기" 메뉴 클릭
- **페이지:** `/manual`

#### Step 5-2: 매뉴얼 학습
**매뉴얼 구성:**
- HDH 솔루션 운영 가이드
- 계좌 개설 방법
- API 연결 방법
- 트레이딩 설정 방법
- 리스크 관리 가이드
- 자주 묻는 질문 (FAQ)

#### Step 5-3: 매뉴얼 숙지 완료
- **결과:** HDH 솔루션 운영을 위한 실무 지식 습득

---

### **[추가 단계] AI 자동화 프로그램 다운로드 및 설정**

#### Step 6-1: AI 프로그램 다운로드
- **액션:** 좌측 사이드바 "AI 프로그램 다운로드" 클릭
- **페이지:** 다운로드 링크 제공
- **다운로드:** AI 자동화 프로그램 설치 파일

#### Step 6-2: 프로그램 설치
- **프로세스:** 설치 파일 실행 → 설치 완료

#### Step 6-3: My Settings 설정
- **액션:** 우측 사이드바 "My Settings" 클릭
- **페이지:** `/settings`

**설정 항목:**
- 계좌 정보 연결
- API 키 입력
- 트레이딩 파라미터 설정
- 알림 설정

#### Step 6-4: 설정 완료 확인
- **결과:** AI 자동화 프로그램 구동 준비 완료

---

### Step 7: HDH 솔루션 운영 시작
- **상태:** 모든 준비 단계 완료
- **표시 메시지:**
```
🎉 HDH 솔루션 운영 준비가 완료되었습니다!

✅ 완료된 단계:
1. ✅ HDH 솔루션 이해 완료
2. ✅ 아카데미 교육 이수 완료
3. ✅ 추천인 코드 발급 완료
4. ✅ 매뉴얼 숙지 완료
5. ✅ AI 프로그램 설치 완료
6. ✅ My Settings 설정 완료

💡 이제 HDH 솔루션 운영을 시작할 수 있습니다!

다음 단계:
- 관리 메뉴에서 수익/손실 관리
- Sunny에게 궁금한 점 문의
```

**결과:** HDH 솔루션 운영 시작

---

## 3. 예외 플로우 (Exception Flows)

### 3-1. 교육 미이수 상태에서 추천인 코드 신청
- **조건:** 아카데미 교육을 이수하지 않은 상태에서 추천인 코드 신청
- **시스템 응답:**
```
❌ 추천인 코드는 아카데미 교육 이수 후 신청 가능합니다.

📌 안내:
- 먼저 아카데미 교육을 수강해주세요.
- 교육 이수 후 추천인 코드 신청이 가능합니다.

[아카데미 교육 신청하기]
```

### 3-2. 수강신청 정보 미입력
- **조건:** 필수 입력 항목을 채우지 않고 제출
- **시스템 응답:**
```
❌ 필수 입력 항목을 모두 채워주세요.
```

### 3-3. API 연결 오류
- **조건:** My Settings에서 잘못된 API 키 입력
- **시스템 응답:**
```
❌ API 연결에 실패했습니다.

- API 키를 다시 확인해주세요.
- 계좌 정보가 정확한지 확인해주세요.

[매뉴얼 보기] [다시 시도]
```

## 5. 성공 기준

### 5-1. 기능적 성공
- ✅ 사용자가 4단계 준비 프로세스를 순서대로 진행할 수 있다
- ✅ 교육 신청이 정상적으로 처리된다
- ✅ 교육 이수 후 추천인 코드가 발급된다
- ✅ AI 프로그램 다운로드 및 설정이 완료된다
- ✅ 모든 준비 완료 시 운영 시작 상태로 전환된다

### 5-2. UX 성공
- ✅ 각 단계의 목적이 명확히 안내된다
- ✅ 다음에 해야 할 행동이 명확하다
- ✅ 진행 상황을 시각적으로 확인할 수 있다
- ✅ 교육 미이수 시 추천인 코드 신청이 차단되고 안내 메시지가 표시된다

### 5-3. 비즈니스 성공
- ✅ 회원이 체계적으로 준비 과정을 완료할 수 있다
- ✅ 추천인 코드를 통해 회원 유입 경로를 추적할 수 있다
- ✅ 교육 이수자만 운영을 시작하도록 관리할 수 있다
