**작성일**: 2025-12-07
**목적**: 실제로 추가할 메뉴를 명확하게 정리

---

## 📌 핵심 요약

현재 우측 사이드바에 있는 **"My 설정"** 섹션을 확장하고, 새로운 위젯 3개를 추가합니다.

---

## 1️⃣ My 설정 메뉴 확장

### 현재 (Before)

```
⚙️ My 설정
├─ 거래 시작일 설정
├─ 누적손익 목표 설정
└─ 계좌 관리 (클릭하면 모달만 열림)
```

### 변경 후 (After)

```
⚙️ My 설정
├─ 거래 시작일 설정
├─ 누적손익 목표 설정
│
├─ 💼 계좌 관리 ▶ (클릭하면 서브메뉴 펼쳐짐)
│  │
│  ├─── 📊 계좌 현황 보기          ← 신규 추가
│  ├─── ➕ 새 계좌 추가            ← 기존 모달
│  ├─── 🔗 연결된 계좌 관리        ← 신규 추가
│  └─── 📜 연동 로그               ← 신규 추가
│
└─ 🔔 알림 설정                    ← 신규 추가
```

### 🎯 목적

- **계좌 관리**를 단순 모달에서 → **4개 서브메뉴**로 확장
- 계좌 상태를 한눈에 볼 수 있는 **"계좌 현황 보기"** 추가
- 연동 오류 확인을 위한 **"연동 로그"** 추가
- 알림을 커스터마이징할 수 있는 **"알림 설정"** 추가

## 🎯 추가하는 메뉴 총정리

### 그룹 1: My 설정 확장

| 메뉴명 | 종류 | 설명 |
|--------|------|------|
| 📊 계좌 현황 보기 | 서브메뉴 | 모든 계좌의 연동 상태를 한 화면에서 확인 |
| 🔗 연결된 계좌 관리 | 서브메뉴 | 연결된 계좌 목록 보기 및 편집/삭제 |
| 📜 연동 로그 | 서브메뉴 | 동기화 이력 및 오류 로그 확인 |
| 🔔 알림 설정 | 메뉴 항목 | 알림 종류별 ON/OFF 설정 |

### 그룹 2: My 대시보드 탭 추가

| 탭명 | 종류 | 설명 |
|------|------|------|
| 계좌별 | 탭 | 4개 계좌의 개별 손익을 프로그레스 바로 표시 |
| 통계 | 탭 | 총 거래 횟수, 승률, 평균 손익 등 통계 |

### 그룹 3: 신규 위젯 추가

| 위젯명 | 종류 | 설명 |
|--------|------|------|
| ⚡ 빠른 액션 | 위젯 | 4개 주요 기능 바로가기 버튼 |
| 🔔 알림 센터 | 위젯 | 최근 알림 3건 + 전체보기 버튼 |
| 🔍 거래 내역 조회 | 위젯 | 필터 + 최근 3건 + 내보내기 버튼 |

## ✅ 구현 우선순위

### 1단계 (가장 중요)

1. **My 설정 → 계좌 관리 서브메뉴 추가**
   - 📊 계좌 현황 보기
   - 🔗 연결된 계좌 관리
   - 📜 연동 로그

2. **My 대시보드 → 계좌별 탭 추가**
   - 4개 계좌 손익 분리 표시
   - 연동 상태 표시 (🟢/🔴)

### 2단계

3. **알림 센터 위젯 추가**
   - 최근 알림 3건 표시

4. **거래 내역 조회 위젯 추가**
   - 필터 + 미리보기 3건

### 3단계

5. **빠른 액션 버튼 추가**
   - 4개 바로가기

6. **통계 탭 추가**
   - 승률, 평균 손익 등

---

## 💡 핵심 개선 포인트

### Before (현재)

- 계좌 관리 → 모달에서만 관리
- 계좌별 손익 → 볼 수 없음 (전체 합산만)
- 알림 → 없음
- 거래 내역 → 달력에서만 일별로 확인

### After (개선 후)

- 계좌 관리 → **4개 서브메뉴로 확장**
- 계좌별 손익 → **탭에서 4개 계좌 분리 표시**
- 알림 → **알림 센터 위젯 추가**
- 거래 내역 → **필터 + 미리보기 + 내보내기**
