## 1. 개요

### 1.1 목적
HDH Fintech 서비스의 효율적인 운영 및 관리를 위한 통합 관리자 대시보드 구축

### 1.2 주요 목표
- 회원 관리 및 등급 체계 관리
- 아카데미 교육 신청 관리
- 추천인 코드 발급 및 관리
- 계좌 및 API 연결 상태 모니터링
- 콘텐츠 관리 (공지사항, 주의사항)
- 고객 지원 관리 (문의, Sunny에게 묻기)
- 관리자 계정 및 권한 관리

### 1.3 관리 대상
- 전체 회원: 약 1,000명
- 운영 회원: 약 847명
- 관리자: 8명
- 총괄 관리자: 2명

## 2. 회원 등급 체계

### 2.1 회원 유형 (4단계)

#### 1️⃣ 일반 회원 (General Member)
- **정의**: HDH 솔루션을 운영하지 않는 회원
- **권한**: 교육 수강 신청, 정보 열람, 커뮤니티 참여
- **제한**: 대시보드 접근 불가, 계좌 연결 불가, AI 프로그램 다운로드 불가
- **배지**: 🔘 회색 "일반" (#94A3B8)
- **승급 조건**: 교육 이수 + 추천인 코드 발급 + 계좌 연결

#### 2️⃣ 운영 회원 (Operating Member)
- **정의**: HDH 솔루션을 실제로 운영하는 회원
- **권한**: 대시보드 접근, 계좌 연결, 거래, AI 프로그램 다운로드
- **보유**: 추천인 코드
- **배지**: 🔵 파란색 "운영" (#3B82F6)
- **강등 조건**: 6개월 이상 미접속 또는 계좌 전체 해제 (관리자 검토 후)

#### 3️⃣ 관리자 (Admin)
- **정의**: 일반 관리 업무 수행자
- **권한**:
  - 어드민 페이지 접근
  - 회원 관리 (일반 ↔ 운영 등급 변경만 가능)
  - 교육/추천인 신청 승인
  - 공지사항/주의사항 작성
  - 고객 문의 응대
- **제한**: 관리자 계정 생성/삭제 불가, 시스템 설정 변경 불가
- **배지**: 🟠 주황색 "관리자" (#F59E0B)
- **임명**: 총괄 관리자만 가능

#### 4️⃣ 총괄 관리자 (Super Admin)
- **정의**: 모든 관리 권한을 보유한 최고 관리자
- **권한**:
  - 모든 관리자 권한 포함
  - 관리자 계정 생성/삭제
  - 모든 회원 등급 변경
  - 시스템 설정 변경
  - 관리자 활동 로그 열람
- **제한**: 최소 1명은 항상 유지 (마지막 총괄 관리자는 등급 변경 불가)
- **배지**: 🔴 빨간색 "총괄" (#EF4444)

### 2.2 등급 변경 규칙

| 변경 경로 | 조건 | 승인 권한 |
|----------|------|----------|
| 일반 → 운영 | 교육 이수 + 추천인 코드 발급 필수 | 관리자, 총괄 관리자 |
| 운영 → 일반 | 계좌 연결 해제 확인 | 관리자, 총괄 관리자 |
| 일반/운영 → 관리자 | 승인 절차 필요 | 총괄 관리자만 |
| 관리자 → 총괄 관리자 | 승인 절차 필요 | 총괄 관리자만 |
| 총괄 관리자 → 관리자 | 최소 1명 유지 필수 | 총괄 관리자만 |

### 3.2 권한 검증 로직

```javascript
// 페이지 접근 권한 체크
function checkPageAccess(userGrade, requiredGrade) {
    const gradeHierarchy = {
        'general': 0,
        'operating': 1,
        'admin': 2,
        'super_admin': 3
    };

    return gradeHierarchy[userGrade] >= gradeHierarchy[requiredGrade];
}

// 등급 변경 권한 체크
function canChangeGrade(adminGrade, targetCurrentGrade, targetNewGrade) {
    if (adminGrade === 'super_admin') {
        return true; // 총괄 관리자는 모든 변경 가능
    }

    if (adminGrade === 'admin') {
        // 일반 관리자는 general ↔ operating만 가능
        const allowedChanges = [
            ['general', 'operating'],
            ['operating', 'general']
        ];

        return allowedChanges.some(([from, to]) =>
            targetCurrentGrade === from && targetNewGrade === to
        );
    }

    return false;
}
```
