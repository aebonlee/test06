## 1. 🎨 테마 설정

### 왜 필요한가?

- **사용자 경험 개선**: 장시간 사용 시 눈의 피로도 감소
- **개인화**: 사용자가 선호하는 스타일 선택
- **전문성**: 다크 모드는 트레이더들이 선호하는 기능

---

### 1.1 화면 모드 (다크/라이트)

#### ⚪ 라이트 모드 (기본)

**현재 상태**: 기본 설정

**특징**:
```
배경색: 흰색 (#FFFFFF)
텍스트: 검정/회색 계열
카드: 밝은 회색 배경
```

**장점**:
- 밝은 환경에서 가독성 좋음
- 데이터 시각화가 선명함

---

#### ⚫ 다크 모드 (신규)

**특징**:
```
배경색: 어두운 회색/검정 (#0F172A, #1E293B)
텍스트: 흰색/밝은 회색
카드: 중간 회색 배경
```

**장점**:
- 야간 사용 시 눈의 피로 감소
- 배터리 절약 (OLED 화면)
- 트레이딩 차트와 어울림

**적용 예시**:
```
현재 (라이트):
┌─────────────────────────┐
│ 배경: 흰색               │
│ 텍스트: 검정             │
│ 카드: 연한 회색          │
└─────────────────────────┘

다크 모드:
┌─────────────────────────┐
│ 배경: 진한 회색          │
│ 텍스트: 흰색             │
│ 카드: 중간 회색          │
└─────────────────────────┘
```

---

#### 🌓 자동 전환

**기능**:
- 시스템 설정을 따라감
- 낮(6시~18시): 라이트 모드
- 밤(18시~6시): 다크 모드

**또는**:
- Windows/Mac 시스템 설정 따라가기

### 1.4 저장 위치

**로컬 저장 (localStorage)**:
```javascript
{
  "theme": {
    "mode": "light",        // "light" | "dark" | "auto"
    "color": "blue",        // "blue" | "green" | "purple" | "red"
    "lastUpdated": "2025-12-07T10:30:00Z"
  }
}
```

**서버 저장 (선택)**:
- 여러 기기에서 동일한 테마 사용
- API: `PUT /api/user/settings/theme`

## 2. 👤 프로필 관리

### 왜 필요한가?

- **개인정보 관리**: 닉네임, 이메일, 전화번호 변경
- **회원 탈퇴**: 서비스 해지
- **프로필 사진**: 개성 표현

---

### 2.1 프로필 정보 항목

#### A. 프로필 사진

**현재 상태**: 아이콘만 표시 (👤)

**개선 후**:
```
┌─────────────────┐
│      [👤]       │  ← 클릭하면 이미지 업로드
│   홍길동님       │
│  Lv.10 회원     │
└─────────────────┘
```

**기능**:
- 이미지 업로드 (JPG, PNG, 최대 2MB)
- 기본 아바타 선택 (10종)
- 이니셜 자동 생성 (예: "홍길동" → "홍")

---

#### B. 닉네임

**현재**: 회원가입 시 설정한 이름

**변경 기능**:
```
닉네임: [홍길동          ]  [변경]

• 2~10자 이내
• 한글, 영문, 숫자 가능
• 특수문자 불가
• 중복 확인 필요
```

**사용 위치**:
- 우측 상단 헤더
- 랭킹 보드
- 커뮤니티 게시글

---

#### C. 이메일

**현재**: 로그인 ID

**변경 기능**:
```
이메일: [hong@example.com  ]  [변경]

• 인증 메일 발송
• 기존 이메일로 확인 코드 받기
• 새 이메일로 인증 완료
```

**주의사항**:
- 로그인 ID가 이메일인 경우 주의
- 변경 시 재로그인 필요

---

#### D. 전화번호

**현재**: SMS 알림용

**변경 기능**:
```
전화번호: [010-1234-5678    ]  [변경]

• SMS 인증 필요
• 새 번호로 인증번호 발송
```

**사용 위치**:
- SMS 알림
- 비밀번호 찾기
- 2단계 인증

---

#### E. 가입일 / 회원 등급

**조회만 가능** (변경 불가):
```
가입일: 2024년 1월 15일
회원 등급: Lv.10 (상위 1%)
총 거래 횟수: 247회
```

### 3.2 테마 설정 JavaScript

```javascript
// 테마 설정 관련 변수
let currentTheme = {
    mode: 'light',
    color: 'blue'
};

// 테마 설정 모달 열기
function openThemeSettingsModal(event) {
    event.stopPropagation();

    // 현재 테마 로드
    loadCurrentTheme();

    document.getElementById('themeSettingsModal').style.display = 'flex';
}

// 테마 설정 모달 닫기
function closeThemeSettingsModal(event) {
    if (event && event.target !== event.currentTarget) return;
    document.getElementById('themeSettingsModal').style.display = 'none';
}

// 현재 테마 로드
function loadCurrentTheme() {
    const savedTheme = localStorage.getItem('hdh_theme');
    if (savedTheme) {
        currentTheme = JSON.parse(savedTheme);
    }
    updatePreview();
}

// 화면 모드 선택
function selectMode(mode) {
    currentTheme.mode = mode;

    // UI 업데이트
    document.querySelectorAll('.mode-option').forEach(option => {
        option.classList.remove('active');
    });
    event.target.closest('.mode-option').classList.add('active');

    updatePreview();
}

// 색상 테마 선택
function selectColor(color) {
    currentTheme.color = color;

    // UI 업데이트
    document.querySelectorAll('.color-option').forEach(option => {
        option.classList.remove('active');
        option.querySelector('.color-check').textContent = '';
    });
    event.target.closest('.color-option').classList.add('active');
    event.target.closest('.color-option').querySelector('.color-check').textContent = '✓';

    updatePreview();
}

// 미리보기 업데이트
function updatePreview() {
    const preview = document.getElementById('themePreview');

    // 다크 모드 적용
    if (currentTheme.mode === 'dark') {
        preview.classList.add('dark-mode');
    } else {
        preview.classList.remove('dark-mode');
    }

    // 색상 테마 적용
    preview.setAttribute('data-color', currentTheme.color);
}

// 테마 저장
function saveThemeSettings() {
    // localStorage에 저장
    localStorage.setItem('hdh_theme', JSON.stringify(currentTheme));

    // 전체 페이지에 테마 적용
    applyTheme();

    // 모달 닫기
    closeThemeSettingsModal();

    alert('테마가 저장되었습니다.');
}

// 테마 적용
function applyTheme() {
    const body = document.body;

    // 다크 모드 적용
    if (currentTheme.mode === 'dark') {
        body.classList.add('dark-mode');
    } else {
        body.classList.remove('dark-mode');
    }

    // 색상 테마 적용
    body.setAttribute('data-theme-color', currentTheme.color);
}

// 페이지 로드 시 테마 적용
document.addEventListener('DOMContentLoaded', function() {
    loadCurrentTheme();
    applyTheme();
});
```

```javascript
function handlePhotoUpload(event) {
    const file = event.target.files[0];

    if (file) {
        // 파일 크기 체크 (2MB)
        if (file.size > 2 * 1024 * 1024) {
            alert('파일 크기는 2MB 이하여야 합니다.');
            return;
        }

        // 이미지 미리보기

    }
```
