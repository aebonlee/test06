## 메뉴별 기능 상세

### 1. 📊 대시보드 개요 (홈)

#### 1-1. 주요 통계 카드 (4개)

**카드 1: 총 회원 수**
- 현재 값: 456명
- 전일 대비 증감: ↑ +3명
- 색상: Primary (골드)

**카드 2: 교육 신청 대기**
- 현재 값: 2건
- 처리 필요 알림
- 색상: Warning (주황)

**카드 3: 추천인 코드 신청 대기**
- 현재 값: 4건
- 처리 필요 알림
- 색상: Info (파랑)

**카드 4: 미처리 문의**
- 현재 값: 5건
- 긴급 알림: ⚠️ 긴급 1건
- 색상: Danger (빨강)

#### 1-2. 최근 활동 타임라인

**실시간 업데이트 항목:**
- 🟢 신규 가입 (예: 김철수) - 15분 전
- 🟡 교육 신청 (예: 이영희) - 1시간 전
- 🔴 긴급 문의 (예: API 연결 오류) - 2시간 전
- 🟢 추천인 코드 신청 (예: 박민수) - 3시간 전

**표시 정보:**
- 활동 아이콘 (상태별 색상)
- 활동 메시지
- 상대 시간 표시

#### 1-3. 긴급 처리 필요 (우선순위 순)

**클릭 가능한 작업 항목:**

| 우선순위 | 작업 내용 | 건수 | 이동 |
|---------|---------|------|------|
| 🔴 High | API 연결 오류 문의 | 3건 | → 문의 관리 |
| 🟡 Medium | 교육 신청 (신규) | 5건 | → 교육 관리 |
| 🟡 Medium | 추천인 코드 신청 | 8건 | → 추천인 코드 관리 |
| 🟢 Low | 일반 문의 | 10건 | → 고객센터 문의 |

**동작:**
- 클릭 시 해당 섹션으로 자동 이동
- Hover 시 카드 강조 효과

#### 1-4. 차트 (2개)

**차트 1: 회원 가입 추이**
- 기간: 최근 7일
- 유형: Line Chart
- 라이브러리: Chart.js

**차트 2: 교육 이수 추이**
- 기간: 최근 4주
- 유형: Bar Chart
- 라이브러리: Chart.js

### 2. 👥 회원 목록

#### 2-1. 회원 테이블

**표시 컬럼:**
- 회원 ID
- 이름
- 이메일
- 전화번호
- 가입일
- 교육 이수 여부
- 추천인 코드
- 상태 (활성/비활성)

**필터 옵션:**
- 교육 이수 여부: 전체 / 이수 / 미이수
- 추천인 코드: 전체 / 발급 / 미발급
- 가입일: 기간 선택

**검색 기능:**
- 이름, 이메일, 전화번호로 검색

**액션:**
- 각 행 클릭 → 회원 상세 페이지

#### 2-2. 회원 상세

**표시 정보:**
- 기본 정보 (이름, 이메일, 전화번호, 닉네임)
- 가입일
- 교육 이수 날짜 (이수한 경우)
- 추천인 코드 (발급된 경우)
- 최근 로그인
- My 누적 손익 요약
- My 손익 저널 건수

**관리자 액션:**
- 교육 이수 처리
- 추천인 코드 발급
- 회원 비활성화

---

### 3. 📚 교육 관리

#### 3-1. 교육 신청 목록

**표시 컬럼:**
- 신청 ID
- 이름
- 이메일
- 전화번호
- 신청일
- 상태 (신청 접수 / 일정 확정 / 교육 진행 중 / 이수 완료)

**필터 옵션:**
- 상태: 전체 / 신청 접수 / 일정 확정 / 교육 진행 중 / 이수 완료
- 신청일: 기간 선택

**액션:**
- 각 행 클릭 → 신청 상세 페이지

#### 3-2. 신청 상세

**표시 정보:**
- 신청자 기본 정보
- 수강 동기
- 현재 트레이딩 경험
- 기대하는 점
- 신청일
- 교육 날짜 (확정된 경우)
- 교육 방식 (온라인/오프라인)

**관리자 액션:**
- 상태 변경: 신청 접수 → 일정 확정
- 교육 날짜 입력
- 교육 방식 선택
- 교육 이수 처리
- 메모 추가

**교육 이수 처리:**
- [이수 처리] 버튼 클릭
- 이수 날짜 자동 입력
- 상태: 교육 진행 중 → 이수 완료
- 자동 트리거:
  - ✅ 이수 완료 이메일 발송
  - ✅ 추천인 코드 신청 자격 부여

---

### 4. 🎯 추천인 코드 관리

#### 4-1. 신청 목록

**표시 컬럼:**
- 신청 ID
- 이름
- 이메일
- 교육 이수 여부 (✅ / ❌)
- 신청일
- 상태 (신청 대기 / 승인 완료 / 거부)
- 추천인 코드 (발급된 경우)

**필터 옵션:**
- 상태: 전체 / 신청 대기 / 승인 완료 / 거부
- 교육 이수 여부: 전체 / 이수 / 미이수

**액션:**
- 각 행 클릭 → 신청 상세 페이지

#### 4-2. 신청 상세

**표시 정보:**
- 신청자 기본 정보
- 교육 이수 여부 및 이수 날짜
- 신청 사유
- HDH 솔루션을 알게 된 경로
- 신청일

**관리자 액션:**

**교육 이수한 경우 (✅):**
- [승인] 버튼 클릭
- 시스템이 자동으로 추천인 코드 생성 (예: HDH2025ABC123)
- 상태: 신청 대기 → 승인 완료
- 자동 이메일 발송

**교육 미이수한 경우 (❌):**
- [거부] 버튼 클릭
- 거부 사유 입력: "교육 미이수"
- 상태: 신청 대기 → 거부
- 거부 사유 이메일 발송

## 주요 관리 작업 흐름

### 흐름 1: 교육 신청 → 이수 처리

```
[교육 관리] → 신청 목록
    ↓
신청 상세 클릭
    ↓
교육 날짜 입력 + 방식 선택
    ↓
상태 변경: "일정 확정"
    ↓
(교육 진행)
    ↓
[이수 처리] 버튼 클릭
    ↓
자동 이메일 발송 + 추천인 코드 신청 자격 부여
```

---

### 흐름 2: 추천인 코드 신청 → 승인

```
[추천인 코드 관리] → 신청 목록
    ↓
신청 상세 클릭
    ↓
교육 이수 여부 확인
    ↓
├─ ✅ 이수 → [승인] 버튼
│   ↓
│   추천인 코드 자동 생성
│   ↓
│   자동 이메일 발송
│
└─ ❌ 미이수 → [거부] 버튼
    ↓
    거부 사유 이메일 발송
```

---

### 흐름 3: 문의 처리

```
[문의 관리] → 미답변 목록
    ↓
문의 상세 클릭
    ↓
답변 작성
    ↓
[답변 전송]
    ↓
자동 이메일 발송 + 상태 변경
```
