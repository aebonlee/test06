## 개요

### 목적
HDH Fintech의 모든 User Flow가 실제 사용자 여정 속에서 어떻게 연결되고 작동하는지 보여줌

### 4대 User Flow 통합
1. **User Flow #1:** 회원가입
2. **User Flow #2:** HDH 운영 준비 및 시작
3. **User Flow #3:** 관리 프로세스 (손익 관리, 설정)
4. **User Flow #4:** 문의 프로세스 (Sunny, 고객센터, 공지)

### 사용자 유형별 여정
- **신규 사용자:** 가입 → 솔루션 이해 → 교육 신청 → 교육 이수
- **준비 중 사용자:** 추천인 코드 신청 → 매뉴얼 학습 → AI 프로그램 설치 → 설정
- **운영 사용자:** HDH 운영 시작 → 손익 기록 → 성과 추적 → 지속 운영
- **장기 사용자:** 안정적 운영 → 수익 증대 → 커뮤니티 참여

- 프롭펌 계좌 운영

💰 수익 구조
- 월 평균 목표 수익률
- 리스크 관리 방법
- 장기 성과 사례

📈 운영 방식
- 계좌 개설
- 자금 운용

#### 1-2. 아카데미 교육 신청 (User Flow #2 - 단계 2)

**진입:**
좌측 사이드바 → "2. 아카데미 교육 수강하기" 클릭

**Step 1: 아카데미 소개 확인**
- 페이지: `/academy-intro`
- 교육 커리큘럼 확인
- 교육 기간 및 방식 안내

**Step 2: 수강신청**
- "수강신청하기" 버튼 클릭
- 페이지: `/academy-apply`

**신청 폼:**
```
이름: [자동 입력]
이메일: [자동 입력]
전화번호: [자동 입력]

수강 동기: [텍스트 입력]
현재 트레이딩 경험: [선택]
  - 없음
  - 1년 미만
  - 1-3년
  - 3년 이상

기대하는 점: [텍스트 입력]

[신청하기]
```

**신청 완료:**
```
✅ 수강신청이 완료되었습니다!

담당자가 확인 후 1-2일 이내에 연락 드리겠습니다.

📌 안내:
- 교육 일정은 개별 안내 예정
- 교육 완료 후 추천인 코드 신청 가능
```
