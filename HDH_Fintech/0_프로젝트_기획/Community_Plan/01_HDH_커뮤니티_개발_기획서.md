## 1. 개요

### 1.1 커뮤니티 정의

**HDH 실전 운영자 커뮤니티**는 HDH Fintech 솔루션을 실제로 운영하는 사용자들이 모여 정보를 공유하고, 경험을 나누며, 서로 성장하는 **폐쇄형 커뮤니티**입니다.

### 1.2 핵심 가치

| 가치 | 설명 |
|------|------|
| **신뢰** | 검증된 운영자들만 참여하는 폐쇄형 커뮤니티 |
| **공유** | 실전 경험과 노하우 공유 |
| **성장** | 함께 배우고 성장하는 환경 |
| **소통** | 운영자 간 원활한 커뮤니케이션 |

### 1.3 접근 권한

```
┌─────────────────────────────────────────────────────────┐
│  커뮤니티 접근 조건                                      │
├─────────────────────────────────────────────────────────┤
│  ✅ HDH Fintech 회원가입 완료                            │
│  ✅ 아카데미 교육 이수 완료                              │
│  ✅ 추천인 코드 발급 완료                                │
│  ✅ 구독 결제 완료 (월 10만원)                           │
└─────────────────────────────────────────────────────────┘
```

**폐쇄형 운영 이유:**
- 검증된 운영자들만의 신뢰 환경 구축
- 민감한 트레이딩 정보 보호
- 커뮤니티 품질 유지

## 2. 주요 기능

### 2.1 게시판 시스템

#### 2.1.1 게시판 구조

```
HDH 실전 운영자 커뮤니티 (6개 게시판)

├── 📢 공지사항
│   └── 커뮤니티 공지 (관리자만 작성)
│
├── 👋 가입인사
│   └── 신규 가입 인사 (전체 회원 작성)
│
├── 🎓 코치 게시판
│   └── 코치 노하우 공유 (코치만 작성)
│
├── ❓ Q&A
│   └── 질문/답변 (전체 회원 작성)
│
├── 📁 정보공유
│   └── 인디게이터, 유용한 사이트, 분석자료 등 (전체 회원 작성)
│
└── 💬 자유게시판
    └── 자유 & 일상, 목표 달성 축하 등 (전체 회원 작성)
```

#### 2.1.2 작성 권한

| 게시판 | 작성 권한 | 읽기 권한 |
|--------|----------|----------|
| 공지사항 | 관리자만 | 전체 회원 |
| 가입인사 | 전체 회원 | 전체 회원 |
| 코치 게시판 | 코치만 | 전체 회원 |
| Q&A | 전체 회원 | 전체 회원 |
| 정보공유 | 전체 회원 | 전체 회원 |
| 자유게시판 | 전체 회원 | 전체 회원 |

#### 2.1.3 회원 유형

| 유형 | 설명 | 권한 |
|------|------|------|
| 관리자 | 시스템 관리자 | 모든 게시판 작성/관리 |
| 코치 | 교육 담당자 | 코치 게시판 작성, 일반 게시판 작성 |
| 일반 회원 | HDH 솔루션 운영자 | 가입인사, Q&A, 자유게시판 작성 |

#### 2.1.4 게시글 기능

| 기능 | 설명 |
|------|------|
| 글쓰기 | 제목, 본문, 이미지/파일 첨부 |
| 댓글 | 댓글, 대댓글 (2단계) |
| 좋아요 | 게시글/댓글 좋아요 |
| 스크랩 | 나중에 볼 게시글 저장 |
| 신고 | 부적절한 게시글 신고 |
| 검색 | 제목/내용/작성자 검색 |

### 2.2 회원 등급 시스템

> **참고:** 수익 인증, 랭킹, 레벨 시스템은 HDH Fintech 웹사이트 본 기능에서 제공됩니다.
> 커뮤니티에서는 웹사이트의 회원 정보를 연동하여 표시합니다.

| 항목 | 표시 방식 | 데이터 출처 |
|------|----------|------------|
| 활동 등급 | 레벨 (Lv.1 ~ Lv.10) | 웹사이트 연동 |
| 영향력 등급 | 순위 (1위 ~ 10위) | 웹사이트 랭킹 연동 |
| 회원 유형 | 코치 / 일반 회원 | 커뮤니티 자체 관리 |

### 2.3 알림 시스템

| 알림 유형 | 설명 |
|----------|------|
| 댓글 알림 | 내 게시글에 댓글이 달렸을 때 |
| 좋아요 알림 | 내 게시글/댓글에 좋아요가 달렸을 때 |
| 멘션 알림 | 다른 사용자가 나를 언급했을 때 |
| 답변 채택 | 내 답변이 채택되었을 때 |
| 공지사항 | 새로운 공지사항이 등록되었을 때 |
| 배지 획득 | 새로운 배지를 획득했을 때 |
