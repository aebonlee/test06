## 1. 비전과 미션

### 🎯 미션 (Mission)

**1. ProActive Senior에게 안정적인 추가 수입 기회 제공**
- 본업을 유지하면서 세컨 인컴(Second Income) 창출
- 검증된 HDH 솔루션으로 월 최소 100만원 ~ 1,000만원 수익 가능
- 감정 트레이딩이 아닌 시스템 기반 안정적 수익 구조

**2. HDH 솔루션 운영을 누구나 쉽게 할 수 있도록 지원**
- 학습부터 실전까지 전 생애주기 완벽 지원
- AI 자동화로 복잡한 과정 간소화
- 체계적인 가이드로 초보자도 즉시 시작 가능

### 🚀 비전 (Vision)

**3년 내 회원 5만명 확보, 그 중 5,000명 이상(10%)을 SUNNY ECONOMIC ECOSYSTEM의 AI 창업과 스타트업 서포터 서비스 회원으로 전환**

- **1년차 목표**: 3,000명 회원 확보
- **2년차 목표**: 15,000명 회원 확보
- **3년차 목표**: 50,000명 회원 확보 + 5,000명 SUNNY ECOSYSTEM 전환
- **궁극적 목표**: HDH Fintech를 SUNNY ECONOMIC ECOSYSTEM의 핵심 입구로 확립

## 2. 사업 개요

### 2.1 사업명

**HDH Fintech** - HDH 솔루션 운영자 지원 플랫폼

### 2.2 사업 분야

- 핀테크 (Fintech)
- 트레이딩 솔루션 운영 지원
- SaaS (Software as a Service)
- 회원제 플랫폼

### 2.3 사업 내용

HDH Fintech는 **HDH 솔루션(프롭펌 헷지 트레이딩 전략)을 운영하는 회원들을 위한 종합 지원 플랫폼**입니다.

- 10년간 검증된 HDH 솔루션 운영 가이드 제공
- AI 자동화 프로그램 제공으로 효율적 운영 지원
- 학습, 준비, 실행, 관리, 자동화 전 과정 통합 지원
- 커뮤니티를 통한 실전 노하우 공유

### 2.4 핵심 특징

**단순 정보 제공 사이트가 아닌 실무 운영 플랫폼**
- ❌ 마케팅/랜딩 페이지 아님
- ✅ 회원 전용 실무 운영 도구
- ✅ 학습 + 실행 + 관리 통합 플랫폼
- ✅ AI 자동화 프로그램 포함

### 2.5 사업 배경

**HDH 솔루션의 검증된 성과**
- 10년간 실전 운영으로 검증된 84:16 헷지 전략
- 무조건 수익이 나는 수학적 구조
- 회원 평균 월 수익: 최소 100만원 ~ 1,000만원

**시장 니즈**
- ProActive Senior의 안정적 추가 수입 니즈
- 감정 트레이딩 실패 경험자들의 체계적 시스템 필요
- 복잡한 트레이딩 프로세스의 자동화 요구

## 3. 서비스 정의

### 3.1 사이트 운영 목적 (3대 핵심)

#### 목적 1: HDH 솔루션 운영 가이드 제공

**회원들이 HDH 솔루션을 실제로 운영할 수 있도록 단계별 가이드 제공**

- 솔루션 학습 → 실전 운영까지 전 과정 지원
- 1차 챌린지 단계 운영 프로세스 안내
- 2차 펀딩계좌 트레이딩 단계 프로세스 안내
- 일일 운영 체크리스트 제공
- 주의사항 및 실시간 알림

#### 목적 2: AI 자동화 프로그램 제공

**복잡한 매매 과정을 AI가 자동으로 처리**

- MT5 연동 자동화 시스템 다운로드 제공
- 설치 및 MT5 연동 가이드
- 사용법 동영상 제공
- 주요 기능:
  - 원클릭 동시 진입 (50ms 이하 동기화)
  - 음성 명령 지원
  - 조건 도달 시 자동 청산
  - SL 자동 감지 및 반대 계좌 즉시 청산 (20-80ms)

#### 목적 3: 회원 유치

**신규 회원을 끌어들이기 위한 플랫폼**

- HDH 솔루션의 가치 전달
- 체계적인 온보딩 프로세스 (준비 4단계)
- 커뮤니티 및 교육 시스템
- 성공 사례 공유 및 동기 부여

### 3.2 사이트 성격

**회원 전용 실무 운영 도구**
- 학습 + 실행 + 관리가 통합된 올인원 플랫폼
- 예비 회원 유치 + 기존 회원 운영 지원 동시 수행
- 전 생애주기 지원 (학습 → 준비 → 실행 → 관리 → 자동화)

## 4. 비즈니스 모델

### 4.1 수익 모델

**월 구독료 (Subscription Model)**

| 항목 | 내용 |
|------|------|
| **구독료** | **월 10만원** |
| 결제 방식 | 월 자동 결제 (카드/계좌이체) |
| 포함 내용 | 모든 서비스 이용 + AI 자동화 프로그램 |

### 4.2 구독료에 포함되는 서비스

**1. 학습 콘텐츠**
- HDH 솔루션 이해하기
- 아카데미 교육 (Day 1-3)
- 헷지 전략 원리, 수익 구조 학습
- 10년 검증 실적 확인
- 매뉴얼 및 가이드 문서

**2. 운영 지원**
- 준비 4단계 온보딩 프로세스
- 1차 챌린지 단계 프로세스 가이드
- 2차 펀딩계좌 트레이딩 단계 프로세스 가이드
- 일일 체크리스트
- 실시간 알림 시스템

**3. AI 자동화 프로그램**
- 프로그램 다운로드
- 설치 및 MT5 연동 가이드
- 사용법 동영상
- 지속적 업데이트

**4. 관리 도구**
- My 설정 (거래 설정, 계좌 관리, API 연결 등)
- My 누적손익 추적
- My 손익 발생일지
- 목표 달성률 확인

**5. 커뮤니티**
- HDH 실전 운영자 커뮤니티 참여
- 레벨 시스템 (Lv.1 ~ Lv.10)
- 연간 누적 순이익 랭킹 Top 10
- 정보 공유 및 Q&A

**6. 고객 지원**
- 고객센터 문의
- Sunny 직접 지원
- 커뮤니티 상호 지원
- 지속적인 매뉴얼 업데이트

### 4.3 구독료의 합리성

**회원 평균 월 수익 대비 저렴한 구독료**

| 구분 | 금액 |
|------|------|
| 회원 평균 월 수익 | 100만원 ~ 1,000만원 |
| 구독료 | 10만원 |
| 수익 대비 비율 | 1% ~ 10% |

**HDH 솔루션의 특징:**
- 무조건 수익이 나는 구조 (84:16 헷지 전략)
- 최소 월 100만원 수익 보장 구조
- 구독료 10만원은 회원 입장에서 부담 없음
- 높은 재구독률 예상

### 4.4 추가 수익원 (향후 검토)

**현재는 월 구독료만 집중**

향후 검토 가능한 추가 수익원:
- 프리미엄 전략 패키지 (고급 AI 전략)
- 1:1 프리미엄 멘토링
- 기업 단체 교육 (B2B)

## 5. 타깃 고객

### 5.1 핵심 타깃

**ProActive Senior 중 안정적 추가 수입을 원하는 사람들**

**ProActive Senior의 정의:**
- 경력 정점에 있는 전문가
- 고령자가 아닌 경험 활용자
- 전문성과 수십년 실무 경험 보유
- 본업을 유지하면서 추가 수입원 모색

### 5.2 타깃 특징

**1차 타깃: 안정적 세컨 인컴을 원하는 직장인/자영업자**
- 본업이 있지만 추가 수입 필요
- 월 100만원 이상 안정적 수익 희망
- 시간적 여유는 제한적 (하루 1-2시간)
- 체계적이고 검증된 시스템 선호

**2차 타깃: 투자 실패 경험이 있는 개인 투자자**
- 코인/선물 거래 경험 있지만 손실 경험
- 감정 트레이딩으로 인한 실패
- 체계적인 시스템과 자동화 필요
- 리스크 관리에 대한 니즈

**3차 타깃: 프롭펌 챌린지 관심자**
- 프롭펌 챌린지에 관심 있지만 시작 방법 모름
- 어디서부터 시작해야 할지 막막
- 검증된 전략과 가이드 필요

### 5.3 타깃 페르소나

**페르소나 1: 김철수 (45세, 회사원)**
- 직장 생활 20년차, 중간 관리자
- 월급 외 추가 수입 필요
- 투자 경험 있지만 손실 경험
- 하루 1-2시간 투자 가능
- 체계적이고 검증된 시스템 선호

**페르소나 2: 이영희 (50세, 자영업자)**
- 작은 사업 운영 중
- 사업 외 안정적 수익원 필요
- 금융 투자 관심 있지만 시간 부족
- AI 자동화 시스템에 매력 느낌
- 커뮤니티 활동 선호

**페르소나 3: 박민수 (38세, 프리랜서)**
- 프리랜서로 수입 불안정
- 안정적 기본 수입 필요
- 선물 거래 경험 있음
- 프롭펌 챌린지 관심
- 실전 검증된 전략 찾음

### 5.4 타깃 규모

**국내 잠재 고객 규모 (추정)**

| 구분 | 추정 규모 |
|------|----------|
| ProActive Senior (40-60대) | 약 500만명 |
| 투자 경험자 | 약 100만명 |
| 프롭펌 챌린지 관심층 | 약 10만명 |
| **초기 타깃 시장** | **약 10만명** |

## 6. 제공 서비스 상세

### 6.1 학습 단계 (Learn)

**"처음이어도 괜찮습니다. 체계적인 교육으로 전문가처럼."**

#### 제공 콘텐츠
- **솔루션 이해하기**
  - HDH의 핵심 원리
  - 10년 검증 실적
  - 84:16 헷지 전략 설명
  - 수익 구조 상세 분석

- **아카데미 교육 (Day 1-3)**
  - Day 1: 솔루션 이론 (프롭펌 챌린지, 금선물 CFD, 헷지 전략)
  - Day 2: 플랫폼 사용법 (MT5 설정, 자동화 시스템)
  - Day 3: 실전 시뮬레이션 (모의거래, 최종 테스트)

- **매뉴얼**
  - 거래 가이드
  - 계좌 설정 방법
  - 비상 대응 매뉴얼

### 6.2 준비 단계 (Prepare)

**"4단계 준비 프로세스로 실전 투입 준비 완료."**

#### 준비 4단계
1. 솔루션 이해하기
2. 아카데미 교육 수강하기 (모의거래 포함)
3. 추천인 코드 받기
4. 매뉴얼 숙지하기

**아카데미 교육 수강:**
- Day 1: 솔루션 이론 (프롭펌 챌린지, 금선물 CFD, 헷지 전략)
- Day 2: 플랫폼 사용법 (MT5 설정, 자동화 시스템)
- Day 3: 실전 시뮬레이션 (모의거래, 최종 테스트)

각 단계마다 상세 가이드 제공, 누락 없이 완벽 준비 가능

### 6.3 실행 단계 (Execute)

**"매일 무엇을 해야 하는지 명확하게. 상세한 프로세스 가이드로 실수 방지."**

#### 1차: 챌린지 단계 프로세스 (5단계)
1. 거래 시간 확인 + 알람 예약
2. 시스템 확인 (MT5, 챌린지/보험금 계좌)
3. USDT 매입 및 송금
4. 챌린지 + 보험금 계좌 동시 거래 실행
5. 일일 결과 확인 및 기록

#### 2차: 펀딩계좌 트레이딩 단계 프로세스
- 펀딩 계좌 2개 확보 시 실행
- 양방향 헷지 후 80% 정산 수령 과정 안내

각 단계마다 **완료/미완료** 상태 실시간 확인 가능

### 6.4 관리 단계 (Manage)

**"설정부터 실적 추적까지, 모든 것을 한 곳에서."**

#### My 설정
- 거래 설정
- 계좌 관리
- API 연결 설정
- 알림 설정
- 데이터 관리
- 프로필 관리

#### My 누적손익
- 월/연 누적 손익 실시간 추적
- 목표 달성률 확인
- 레벨 및 랭킹 시스템

#### My 손익 발생일지
- 일자별 거래 기록
- 수익/손실 추적
- 패턴 분석 및 개선점 도출

### 6.5 자동화 단계 (Automate)

**"거래 실행을 AI가 대신. 정확도는 높이고, 시간은 절약."**

#### AI 자동화 시스템 v1.1 주요 기능

**1차 챌린지 단계 자동화:**
- 원클릭 동시 진입: 챌린지 + 보험금 계좌 1회 클릭으로 동시 체결 (50ms 이하)
- 음성 명령 지원: "매수 0.1랏" 등 음성으로 즉시 매매
- 조건 도달 시 자동 청산: TP/SL 도달 시 자동 청산

**2차 펀딩 트레이딩 단계 자동화:**
- SL 자동 감지: 펀딩 계좌 A/B의 손절 발생 실시간 모니터링
- 반대 계좌 즉시 청산: 한쪽 손절 감지 시 반대 계좌 자동 시장가 청산
